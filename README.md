# tim

- tim is basically vim but worse : (

- currently working on making it better

- this is just one of those attempts to reinvent the wheel for fun

### Build Instructions

```bash
cmake -B build
cmake --build build
```

### Run Instructions

```bash
./build/tim <file>
```
